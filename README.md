### 🚀 Highlighted Project

#### **FaceFortify 🔐**
*A cross-platform, zero-knowledge desktop folder-locking app featuring advanced facial recognition and enterprise-grade encryption.*

*   **Offline Biometrics:** Built liveness-detection and facial verification using **ONNX Runtime (ArcFace)** and **OpenCV Sharp** with 99%+ verification accuracy.
*   **Zero-Knowledge Encryption:** Implemented client-side **AES-256-CBC** file/folder encryption with hardware-bound keys derived via **PBKDF2** (100,000 iterations).
*   **Modern UI:** Developed a responsive and fluid interface using **Avalonia UI** following the MVVM pattern and ReactiveUI.
*   **OS-Level Lock:** Native system security mapping using Windows ACL and Linux `chmod` file permission management.
*   *Tech Stack:* `.NET 8`, `C#`, `Avalonia UI`, `ONNX Runtime`, `OpenCV Sharp`, `AES-256`, `PBKDF2`.
